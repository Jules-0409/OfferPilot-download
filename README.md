## 它能帮你做什么

1. **简历优化** —— 针对每个岗位 JD,分析匹配度、定制这份简历、指出怎么改。**只优化你真实经历的表达,绝不编造**;需要补的内容只作"待你确认的建议",不会自动写进简历。
2. **半自动投递** —— 在 Boss 直聘建立联系、发定制话术。**人工确认 + 限频**,像真人,不无脑海投。
3. **精准沟通** —— 自动检测 HR 新回复,生成贴合岗位的回复草稿,你审一眼再发。
4. **追踪复盘** —— 投了哪些、回复率、邀约/面试状态、哪类岗位回复率高,一目了然。

## 下载哪个?

到 **[最新 Release 页面](../../releases/latest)** 下载对应你系统的文件:

| 你的系统 | 下载这个文件 |
|---|---|
| **Windows**(10/11) | **`OfferPilot-Windows.exe`** |
| **Mac**(Apple Silicon · M1/M2/M3/M4 芯片) | **`OfferPilot-mac-AppleSilicon.zip`** |
| Intel 芯片的 Mac | 暂不支持(需要的话联系作者) |

桌面版**自带 Python 和所有依赖,无需预装任何东西**;只要电脑上有 **Chrome 或 Edge** 浏览器即可(Windows 自带 Edge,Mac 建议装 Chrome)。

## 安装与首次启动

### Windows
1. 下载 `OfferPilot-Windows.exe`,双击运行。
2. 首次可能弹 **「Windows 已保护你的电脑」**(因为还没买代码签名证书)→ 点 **更多信息** → **仍要运行**。
3. 稍等十几秒,浏览器会自动打开 OfferPilot 界面。

### Mac(Apple Silicon)
1. 下载 `OfferPilot-mac-AppleSilicon.zip`,双击解压得到 **OfferPilot.app**,拖进「应用程序」文件夹。
2. 首次打开会提示 **「无法验证开发者」/「已损坏」**(同样因为未签名)。二选一绕过:
   - **图形方式**:双击一次(会被拦)→ 打开 **系统设置 → 隐私与安全性** → 滑到最下面,点 **「仍要打开」** → 再确认一次。
   - **终端一行(更省事)**:打开「终端」,粘贴回车:
     ```bash
     xattr -dr com.apple.quarantine /Applications/OfferPilot.app
     ```
     之后双击就能正常打开。
3. 浏览器会自动打开 OfferPilot 界面。

> 启动后如果浏览器没自动弹出,手动打开 **http://127.0.0.1:8000**(若被占用会自动换 8010 等端口,看启动窗口的提示)。

## 准备 DeepSeek API Key

桌面版用**你自己的** DeepSeek Key(便宜、可控、不和别人共用额度)。

1. 打开 <https://platform.deepseek.com>,注册/登录。
2. 进 **API Keys** → **创建 API Key**,复制那串以 `sk-` 开头的内容(只显示一次,记得存好)。
3. 确保账户有一点余额(充值入口在控制台;分析/话术消耗很小)。
4. 回到 OfferPilot 界面,顶部蓝色横幅里**粘贴 Key → 保存**。以后想换,点右上角 **🔑 API Key**。

> Key 只保存在你本机(`~/.offerpilot/config.json`),**不会上传任何服务器**。

## 完整使用流程

1. **导入简历**:进「简历库」,上传你的简历(PDF / Word / 文本)或直接粘贴。
2. **打开隔离浏览器**:进「岗位匹配分析」,点 **「打开隔离 Boss 浏览器」**。会弹出一个**独立的 Chrome/Edge 窗口**(独立 profile,不影响你平时用的浏览器)。
3. **登录 Boss**:在那个独立窗口里登录你的 Boss 直聘账号,打开搜索页或某个岗位详情页。
4. **读取 + 分析**:回到 OfferPilot,点「读取当前页岗位列表」或「读取当前 Boss 岗位」→ 自动算匹配分、列出哪条要求达标/不达标。
5. **定制简历 / 话术**:进「简历定制」「沟通话术」,针对该岗位生成定制内容(不编造,缺口标为待确认)。
6. **投递**:在「一键投递」页确认后投递。系统会建立联系并发出定制话术,**投前人工确认 + 限频**。
7. **接住回复**:HR 回复后,「自动回复」页能检测到并生成回复草稿,你审一眼再发。
8. **追踪复盘**:「追踪复盘」页看回复率、邀约/面试、岗位方向趋势。

> 全程那个 Boss 浏览器都在你眼前,你能随时看到它在做什么、随时接管。

## 为什么浏览器要在你自己电脑上跑

Boss 直聘按 **IP** 风控。如果大家共用一台云服务器上的浏览器,所有账号都从**同一个数据中心 IP** 出去,很快会被识别、触发人机验证甚至封号。

把隔离浏览器放在**你自己的电脑和家庭网络**里、用**你自己的登录态**,是最贴近真人、最不容易被风控的方式 —— 这也是桌面版存在的原因。

## 安全与隐私

- 你的 **Boss 登录、Cookie、聊天记录只留在本机**,不上传任何服务器。
- **DeepSeek Key** 存在本机 `~/.offerpilot/config.json`。
- **不编造经历**:简历优化只重组/强化/量化你的真实经历;任何"新增事实"只作为"待你确认的建议",绝不自动写入简历。
- **限频 + 人工确认**:默认像真人一样有发送间隔和每日上限,投递/发消息前要你确认。

## 常见问题

**Q:双击没反应 / 闪退?**
确认装了 Chrome 或 Edge。Windows 上若被杀毒软件拦截,把 exe 加入信任名单。仍不行就把启动窗口里的报错发给作者。

**Q:Mac 提示「OfferPilot 已损坏,无法打开」?**
这是未签名 App 被 Gatekeeper 拦了,不是真的损坏。用上面的 `xattr -dr com.apple.quarantine ...` 一行命令即可解决。

**Q:投着投着 Boss 弹出"人机验证"/滑块?**
这是 Boss 的正常风控。软件会**自动暂停**并提示你 —— 切到那个独立 Boss 浏览器窗口,手动拖一下滑块完成验证,再回来继续投剩下的。

**Q:端口被占用?**
软件会自动从 8000 往后找空端口(8010/8020…)。看启动窗口提示的地址,或手动访问对应端口。

**Q:能多平台吗(前程无忧/猎聘)?**
目前只做 Boss 直聘。

**Q:会不会被封号?**
本工具刻意"精投 + 限频 + 人工确认 + 用你自己的 IP",就是为了贴近真人、降低风险;但任何自动化都无法 100% 保证,请理性使用、量力而行。
