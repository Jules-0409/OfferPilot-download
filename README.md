## ⬇️ 下载桌面版(Windows)

1. 到 **[Releases 页面](../../releases/latest)** 下载最新的 **`OfferPilot.exe`**。
2. 双击运行。首次 Windows 可能弹「Windows 已保护你的电脑」→ 点 **更多信息** → **仍要运行**(因为暂未做代码签名)。
3. 浏览器会自动打开,按提示**填入你自己的 DeepSeek API Key**(去 <https://platform.deepseek.com> 注册创建,Key 只存在你本机、不上传)。
4. 在「岗位匹配分析」页点 **打开隔离 Boss 浏览器** → 在弹出的独立 Chrome/Edge 窗口里登录 Boss → 回到界面读取岗位、分析、投递。

> 无需预装 Python 或任何东西;电脑上有 **Chrome 或 Edge** 即可(Win10/11 自带 Edge)。

## 为什么浏览器要在你自己电脑上跑?

Boss 直聘按 IP 风控。如果大家共用一台云服务器的浏览器,所有账号从同一个数据中心 IP 出去,很快会被风控甚至封号。所以隔离浏览器跑在**你自己的网络和电脑**上,用你自己的登录态——这是最安全的方式。

## 安全与隐私

- 你的 Boss 登录、Cookie、聊天记录**只留在你本机**,不上传任何服务器。
- DeepSeek Key 保存在本机 `~/.offerpilot/config.json`。
- 不编造简历经历:只优化真实经历的表达,新增事实只作"待你确认的建议",绝不自动写入。

## 常见问题

- **双击没反应**:确认装了 Chrome 或 Edge;杀毒软件误拦时把 exe 加入信任。
- **触发 Boss 人机验证**:正常风控,软件会自动暂停并提示你去隔离浏览器手动拖滑块,完成后回来继续。
- **想换 Key**:界面右上角「🔑 API Key」随时可改。
